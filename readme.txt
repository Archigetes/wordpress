## How to install

* Unzip and then copy the cats456 directory to /wp-content/themes/ in your WordPress installation. 
* Activate from admin menu. 

## How to use

* Theme options placed at "Appearance >> Customize"
* Widgets can be found at "Appearance >> Widgets"
* Documentation: http://wpblogs.ru/themes/how-to-video-sg-window-theme/
* cats456 works with WordPress 4.2 and higher. Be sure you use latest version of WordPress.
